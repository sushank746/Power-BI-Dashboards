# Power BI Dashboards

This repository contains Power BI dashboards created for analyzing different business datasets.
The projects focus on data visualization, KPI analysis, and generating business insights using Power BI.

## Projects

### 1. Blinkit Sales Analytics Dashboard

This dashboard analyzes sales performance inspired by the Blinkit business model.
It includes KPIs such as Total Sales, Average Sales, and Average Rating.
The dashboard also shows product category distribution, outlet size comparison, and outlet location analysis using interactive filters and slicers.

### 2. IT Helpdesk Performance Dashboard

This dashboard analyzes IT support ticket data to monitor helpdesk performance.
It tracks metrics like total tickets, open and closed tickets, resolution time, and ticket priority distribution.
The dashboard helps understand support workload and service efficiency.

### 3. Global Retail Sales Dashboard

This dashboard provides insights into retail sales performance.
It includes KPI cards for revenue, profit, and orders along with regional sales analysis and category-wise performance.
Interactive filters allow users to explore the data dynamically.

## Tools Used

* Power BI
* Power Query
* DAX
* CSV datasets
